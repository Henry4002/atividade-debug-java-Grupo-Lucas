# Pedra, Papel e Tesoura (Java)

Este projeto é um programa simples em Java que simula o jogo Pedra, Papel e Tesoura entre o usuário e o computador.

O sistema recebe a escolha do usuário, gera uma escolha aleatória para o computador e determina o resultado da partida com base nas regras do jogo.

---

## Funcionalidades
- Entrada de dados pelo usuário
- Geração de jogada aleatória pelo computador
- Comparação das jogadas
- Exibição do resultado (vitória, derrota ou empate)

---

## Entrada
O programa espera que o usuário digite uma das seguintes opções:

- `pedra`
- `papel`
- `tesoura`

A entrada não diferencia maiúsculas de minúsculas.

---

## Saída
O programa retorna:

- A escolha do computador
- O resultado da partida:
  - `Empate`
  - `Você venceu`
  - `Você perdeu`

---

## Regras do jogo
- Pedra vence tesoura
- Tesoura vence papel
- Papel vence pedra
